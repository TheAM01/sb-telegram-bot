LLedger Bots Admin

Run your own payment ledger bot

Submit your Telegram bot token below — no account needed. Once an administrator approves it, your bot goes live running the shared ledger code: running totals, payment details, the same commands for every bot.

Only these Telegram user IDs will be able to use your bot. Don't know a user's ID? Have them message /myid in the group, or use a bot like @userinfobot.

How it works

1

Create a bot

Message @BotFather on Telegram, run /newbot, and copy the token it gives you.

2

Submit the token

Paste it above, along with the user IDs you want to allow (optional). We verify the token with Telegram and queue it for review. Check back any time with the same token.

3

Get approved

An administrator reviews the submission. Once approved, your bot is live within seconds — add it to your group and go.

What your bot will do

CommandDescription
<amount>A message that is just a number is booked as an entry against the group's running total. 500 and +500 add; -500 subtracts. Decimals like 1500.50 work.
<calculation>A price can be a sum instead of a number: 5+2 books 7, 5*7 books 35, (100-40)/2 books 30. Supports + - * / (and ÷ / ×); the result is shown in the reply.
<any message>The last line of any message — of any length — is read as the price and applied to the group's running total; the lines above it are ignored. The price must be a bare number or a calculation: if the last line has any text with it (e.g. 10800 CP, which is COD points), the message is ignored. Works on photo captions too.
0Show the current remaining amount.
/paidMark the running total as paid and reset it to 0.
/setpayment <method> <details>Save or update a payment method for the chat, e.g. /setpayment binance 123456789.
/paymentShow all saved payment details for the chat. /payments works too.
/delpayment <method>Remove a saved payment method.
/myidReply with your Telegram user ID — use it to fill in the allowed-users list above. Works even if you're not on the list yet.
Your token is verified with Telegram and stored only so your bot can be started for you. Each bot's data — including its allowed-user list — is isolated from every other bot.
